Главное

  • 27.08.2021
    • Возобновление разработки утилиты update-ports, предназначенной для обновления системы портов Calmira. Новая версия утилиты переписана с нуля на Python. Из нововведений:
      1. Проверки на коректность скачивания, распаковки и установки портов;
      2. Теперь скрипт можно запустить только от имени root;
      3. Немного изменён синтаксис утилиты
  • 24.08.2021
    • Прекращение поддержки оригинального cpkg и начало разработки cpkg2, нацеленного на более быструю и стабильную работу, а так же упрощённую работу с базой данных
  • 21.08.2021
    • Начало масштабного редизайна сайта дистрибутива. Новый дизайн будет включать в себя простой и понятный внешний вид сайта, будут добавлены страницы для скачивания дистрибутива, страницы с информацией о каждом релизе дистрибутива, а так же страницы некоторых компонентов Calmira (таких как CABS, cpkg и пр.).
  • 20.08.2021
    • Удаление из портов рабочего окружения GNOME Shell ввиду его несовместимости с Calmira;
  • 16.08.2021
    • Добавление скрипта makesystem, служащего для автоматизации сборки системы. Входит в состав CABS.
  • 15.08.2021
    • Добавление скрипта mkiso, служащего для автоматизации сборки системы в загрузочный iso образ. Входит в состав CABS (Calmira Automated Build System) - автоматизированной системы сборки Calmira.

Выход релизов системы и других компонентов

  • 24.08.2021
    • Начало разработки cpkg2;
  • 21.08.2021
    • Релиз первого прототипа стабильного релиза Calmira LX4 1.1
